This app is to help anyone playing "Animal Crossing: New Horizons" keep track of the large number of in game collectibles. If you are in Redd's ship and don't know if you already own a certain painting, use this app instead of running to the museum to check! Same goes for K.K. Slider songs and more!

Adding more collections soon! Look out for Bugs, Sea Creatures, Fish, Housewares and more!
